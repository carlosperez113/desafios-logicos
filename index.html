Solución: Números Primos

Explicación de la Lógica

La función verifica si un número es primo. Primero, descarta casos obvios (menores a 2, o múltiplos de 2 y 3). Luego, itera desde 5, saltando de 6 en 6, lo que permite comprobar dos posibles divisores (`i` e `i+2`) en cada paso. Este método es una optimización del Criba de Eratóstenes para un solo número.

Código de Ejemplo

function isPrime(num) {
  if (num <= 1) return false;
  if (num <= 3) return true;
  if (num % 2 === 0 || num % 3 === 0) return false;
  
  for (let i = 5; i * i <= num; i = i + 6) {
    if (num % i === 0 || num % (i + 2) === 0) {
      return false;
    }
  }
  return true;
}

Solución: Serie Fibonacci

Explicación de la Lógica

Se utiliza un enfoque iterativo para mayor eficiencia. Se inicia un array con `[0, 1]`. Luego, un bucle `while` calcula el siguiente número sumando los dos últimos de la serie. Si el nuevo número no excede el límite, se añade al array. Este método evita la sobrecarga de la recursividad.

Código de Ejemplo

function fibonacciSeries(limit) {
  if (limit < 0) return [];
  if (limit === 0) return [0];

  const series = [0, 1];
  while (true) {
    let next = series[series.length - 1] + series[series.length - 2];
    if (next > limit) break;
    series.push(next);
  }
  return series;
}

Solución: Cálculo de IMC

Explicación de la Fórmula

El cálculo sigue dos pasos: 1. Conversión: Se convierte el peso de libras a kilogramos (`pesoLibras * 0.453592`). 2. Fórmula IMC: Se aplica la fórmula estándar `peso (kg) / (altura (m)²)`. Finalmente, el resultado numérico se clasifica en categorías (Bajo peso, Normal, etc.) para una mejor interpretación.

Código de Ejemplo

function calculateBmi(weightLbs, heightM) {
  const weightKg = weightLbs * 0.453592;
  const bmi = weightKg / (heightM ** 2);
  let category;

  if (bmi < 18.5) category = 'Bajo peso';
  else if (bmi < 24.9) category = 'Peso normal';
  else if (bmi < 29.9) category = 'Sobrepeso';
  else category = 'Obesidad';
  
  return `IMC: ${bmi.toFixed(2)} (${category})`;
}

Solución: Palabra Palíndroma

Explicación de la Lógica

Para una comparación robusta, la cadena de texto se "normaliza": 1. Se convierte a minúsculas. 2. Se eliminan todos los caracteres que no sean letras o números (espacios, signos de puntuación) usando una expresión regular. Luego, se compara la cadena limpia con su versión invertida. Si son idénticas, es un palíndromo.

Código de Ejemplo

function isPalindrome(text) {
  // Normaliza el texto: a minúsculas y sin caracteres no alfanuméricos.
  const cleaned = text.toLowerCase().replace(/[\W_]/g, '');

  // Invierte la cadena limpia.
  const reversed = cleaned.split('').reverse().join('');
  
  // Compara la versión limpia con la invertida.
  return cleaned === reversed;
}
